【 概要 】
チラシの裏のような感覚で思いついたことやちょっとメモっておきたいことなどを
とりあえず書き留めておくためのメモ帳です。煩わしい読み込みや保存の作業が必要ないので
ちょっとメモをとりたいときは、パッと開いて書き込んでおいて閉じれば保存されます。
 チラシの裏のような感じで使い捨てのメモを取りたいときにご使用ください。

【 インストール 】
適切な解凍ソフトを使用して、任意のフォルダに解凍します。
※バージョンアップについて
「チラシの裏」を解凍したフォルダに新バージョンの「chirashi.exe」のみを上書きコピー
して下さい。

【 アンインストール 】
chirashi.exe
memo.txt
cfg.ini
ReadMe.txt
を削除してください。
*レジストリは使用していません。

【 動作確認環境 】
OS
Windows 2000
Windows XP

Windows 98,Windows Me でも動作しますがメモコーンポーネントを使用してるので32kb制限があります。


【 簡単な使用方法 】
chirashi.exeをダブルクリックして起動してください。メモを書き込んで閉じれば保存されます。
新たにメモを書きはじめたいときは、ツールボタンの一番左「次のメモ」ボタンを押せば
仕切り線が挿入され新たなメモを書きやすくなります。
